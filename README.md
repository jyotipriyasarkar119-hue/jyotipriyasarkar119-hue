<h2>🚀 Highlighted Project</h2>

<h3>🔐 Comparative Analysis of Spatial Mixing and Probabilistic Debiasing for SRAM-Based Random Bit Generation</h3>

<p><b>Objective:</b><br>
To evaluate entropy enhancement techniques applied to SRAM startup data and analyze their statistical robustness.
</p>

<h4>Key Contributions:</h4>
<ul>
<li>Implemented raw SRAM bit extraction</li>
<li>Designed XOR-based spatial mixing model</li>
<li>Applied probabilistic debiasing methods</li>
<li>Computed Shannon entropy per run</li>
<li>Measured bias deviation and frequency stability</li>
<li>Generated statistical comparison graphs:
    <ul>
        <li>Bit Count vs Run</li>
        <li>Bias Deviation per Run</li>
        <li>Frequency vs Shannon Entropy</li>
        <li>Bit Rate vs Entropy</li>
    </ul>
</li>
</ul>
